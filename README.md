# Sports Web App

Streamlit web app to compare cricket players and predict performance using ML.

## Features
- Upload player CSV
- Compare two players with charts
- Predict existing player performance
- Predict new player performance
- Top player rankings with visualization

## How to run
1. Clone the repository:
   git clone https://github.com/saisree1925/sports-web-app.git
2. Navigate into folder:
   cd sports-web-app
3. Install required packages:
   pip install -r requirements.txt
4. Run the app:
   streamlit run app.py
5. Upload CSV in the app and start using it
